# Wine clustering (PCA + KMeans)

Unsupervised clustering of the **sklearn Wine dataset** (13 chemical features, 178 samples).
The workflow includes data collection and cleaning, MinMax scaling, PCA (2D projection), KMeans clustering, visualization in PCA space.

> Note: The notebook is written in Croatian. This README provides an English summary.

## How to run
```bash
pip install -r requirements.txt
jupyter notebook wine_clustering.ipynb
